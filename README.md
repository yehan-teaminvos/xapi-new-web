# Xapi Web

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

setup your .env file

```txt
NEXT_PUBLIC_CHAT_URL=https://...
NEXT_PUBLIC_CHAT_KEY=xxx
NEXT_PUBLIC_SIGN_IN_URL=
NEXT_PUBLIC_SIGN_UP_URL=
NEXT_PUBLIC_DOC_URL=
NEXT_PUBLIC_EDITOR_URL=
```
